# Examples - Rich Text Builder

1. Create a sheet named `tags`.
2. Paste `tags_sample.csv` starting at A1.
   - Column A = Title (required)
   - Column B = Sub text 1 (optional)
   - Column C = Sub text 2 (optional)
3. Create a sheet named `gen_view`.
4. Run `generateRichText()` or use menu -> Generator -> Start.
5. Check cells A22, A25, A28, A31 in `gen_view` for the formatted blocks.
