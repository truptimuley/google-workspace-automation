# Module 08 — Drive Utilities

Small Apps Script tools for common Drive tasks: list files in a folder, move/rename files in bulk, resolve and fix shortcuts, and download files metadata.

Use these when you need quick Drive housekeeping without switching to the web UI.

---

## Files

- `listFiles.gs` — list files in a folder to a sheet
- `moveRename.gs` — move or rename files using a control sheet
- `shortcutHandler.gs` — resolve Drive shortcuts to their targets
- `bulkDownload.gs` — create a CSV of file metadata (for audit)
- `utils.gs` — helpers

`examples/`
- `folder-files-sample.csv` — example output
- `README.md` — quick steps
